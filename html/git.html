Git

  • cd - используется для перехода к папке.
  • git init - используется для создания репозитория.
  • git clone - используется для клонирования удаленного репозитория с GitHub на компьютер.
  • git status - используется для отслеживания произошедших изменений.
  • git add - используется для добавление измененного файла в отслеживаемые.
